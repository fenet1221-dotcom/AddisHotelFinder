# AddisHotelFinder

## Project Description

AddisHotelFinder is a hotel booking web application designed to help users find and reserve hotels in Addis Ababa.

The system allows customers to view available hotels, register, login, and make room bookings. Administrators can manage hotels, rooms, users, and bookings.

## Technology Stack

### Backend
- Node.js
- Express.js
- PostgreSQL

### Frontend
- React (to be implemented)

### Security
- JWT authentication
- Password hashing

## Current Features

- Backend server setup
- PostgreSQL database connection
- REST API foundation
